Architecture Decision Records
=============================

Records of significant architectural decisions taken in voxy-linux. Each
ADR captures the context, the chosen option, the alternatives we
rejected, and the consequences we accept.

Format follows `MADR <https://adr.github.io/madr/>`_ lightly. Filenames
are ``NNNN-kebab-title.{rst,md}`` — the four-digit prefix gives lexical
ordering and a stable URL slug.
